# SafeHands Web Application

This is the official web platform for SafeHands, built with React, Vite, and Tailwind CSS.

## Getting Started

1.  **Install Dependencies:**
    ```bash
    npm install
    ```

2.  **Run Development Server:**
    ```bash
    npm run dev
    ```

3.  **Build for Production:**
    ```bash
    npm run build
    ```

## Project Structure

-   `src/App.jsx`: Main application logic, routing, and pages.
-   `src/index.css`: Global styles and Tailwind directives.
-   `public/`: Static assets (images, logos).

## Customization

-   **Theme:** Update `tailwind.config.js` to change colors or animations.
-   **Content:** Edit constants (BRAND, SERVICES, PRICING) in `App.jsx`.
